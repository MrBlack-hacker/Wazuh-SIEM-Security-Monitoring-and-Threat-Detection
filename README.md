# Wazuh SIEM Security Monitoring and Threat Detection

## Project Overview

This project demonstrates the implementation of a Security Information and Event Management (SIEM) solution using Wazuh for centralized log collection, security monitoring, and threat detection.

## Lab Environment

* Ubuntu Linux (Wazuh Server)
* Windows Server 2022 (Target Machine)
* Kali Linux (Attacker Machine)
* Sysmon
* VirtualBox

## Technologies Used

* Wazuh SIEM
* Sysmon
* Windows Server 2022
* Ubuntu Linux
* Kali Linux
* Nmap
* Hydra
* PowerShell

## Attack Simulations

### Nmap Reconnaissance Detection

Network scanning activities were performed using Nmap and successfully detected by Wazuh.

### Hydra Brute Force Detection

Multiple failed authentication attempts generated by Hydra were detected and correlated by Wazuh.

### PowerShell Activity Monitoring

PowerShell commands executed on the Windows endpoint were collected and analyzed through Wazuh.
